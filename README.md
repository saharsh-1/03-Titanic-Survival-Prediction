# Titanic Survival Prediction

## Overview
Predict survival of passengers using machine learning.

## Steps
- Data Cleaning
- Feature Encoding
- Model Training

## Models Used
- Logistic Regression (~75% accuracy)
- Decision Tree (~80% accuracy)

## Evaluation
- Accuracy
- Confusion Matrix
- Precision & Recall

## Key Insight
Decision Tree performed better in identifying survivors but had slightly higher false positives.
